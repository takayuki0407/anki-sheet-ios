# 中間チェックポイント

セッション中の重要な決定・発見を今すぐ保存します。

## 実行内容

### 1. CLAUDE.md の即時更新
直前までの作業で確定した内容を反映する：
- 設計決定事項の追記
- 発見した課題の追記
- 解決済み課題の移動

### 2. 現在の調査・実装結果を docs/research/ または docs/tasks.md に保存

### 3. 現在地の報告

以下の形式で状況を報告する：

```
【チェックポイント】

📍 現在地
[今どこまで進んでいるか]

✅ ここまでに確定したこと
- [箇条書き]

🔄 作業中（未完了）
- [箇条書き]

⚠️ 気になっている点（要確認）
- [箇条書き]（なければ「なし」）
```

### 4. /compact の提案

コンテキスト使用量が多い場合は、この後 /compact を実行することを提案する。
